## Phase 1: Automation Thinking

### Week 1: How Businesses Actually Work
1. **Business Anatomy** — Map any company's operations: Sales → Ops → Support → Finance → Growth
2. **Identifying Waste** — Where companies bleed time and money — the 80/20 of operational waste
3. **Automation vs AI vs Software** — Clear distinction: when to use automation, when AI, when custom software
4. **The Automation Opportunity Canvas** — Repeatable framework to audit any business and find Rs.5L+ of automation opportunities
5. **The Agentic Loop: Sense → Think → Decide → Act → Learn** — The mental model behind every system in this bootcamp
6. **Speaking Business Language** — How to talk to business owners about automation in outcomes, not tech

**Project:** Automation audit of a real business — identify 10+ automatable processes with ROI estimates
**Tools:** Whiteboard, Excalidraw, Google Docs

### Week 2: Designing Systems, Not Workflows
1. **Universal Automation Pattern** — Input → Process → Output → Feedback Loop
2. **The Agentic Loop** — Applied: Sense → Think → Decide → Act → Learn
3. **Event-Driven Thinking** — Triggers, conditions, actions, and error handling as a design language
4. **State Machines in Business** — How leads, tickets, and orders change states — automate transitions
5. **Data Flow Architecture** — Where data lives, how it moves, how it transforms across systems
6. **Cost-Benefit Analysis** — Is this worth automating? Framework for deciding what to build first

**Project:** Design a complete lead-to-close system on paper — trigger map, data flow, Agentic Loop applied
**Tools:** Excalidraw, Draw.io, pen & paper

Build a full-stack AI Personal Assistant — backend, frontend, dashboard — that schedules work without being asked, nudges decisions, monitors systems, and alerts you when things need attention. Use it daily. Sell it to clients for Rs.1L-5L per deployment.

### Week 15: Backend — AI Brain, Memory & Agentic Core
1. **AI Brain Architecture** — Context management, persistent memory, goal tracking — Agentic Loop in a product
2. **Integration Layer** — Connect Gmail, Google Calendar, ClickUp/Notion, LinkedIn APIs
3. **Tool Calling System** — AI decides actions: send email, create task, post on LinkedIn, schedule meeting
4. **Proactive Intelligence** — AI monitors calendar, deadlines, inbox — acts before you ask
5. **Personal Knowledge Base** — AI knows about YOU — projects, preferences, writing style, schedule
6. **API Design & Webhook Receivers** — Endpoints for frontend + real-time triggers

**Project:** Working AI backend: reads/sends emails, checks calendar, creates tasks, nudges decisions, posts on LinkedIn — proactively
**Tools:** FastAPI/Node.js (AI-generated), LLM APIs, Gmail API, Calendar API, LinkedIn API, Vector DB

### Week 16: Frontend — Dashboard, Chat & Operator Control Center
1. **AI Chat Interface** — Real-time conversation with your AI operator — actual task execution
2. **Operator Dashboard** — Today's priorities, unread alerts, upcoming meetings, pending decisions — surfaced by AI
3. **Action Center** — Trigger automations, approve AI suggestions, review and edit drafts
4. **System Monitor View** — Track which automations ran, what failed, what needs attention
5. **Smart Notifications** — AI proactively surfaces what matters — not everything, just what needs you
6. **Mobile-Responsive Design** — Works on desktop, tablet, and mobile as a PWA

**Project:** Functional dashboard: AI chat + task overview + email summary + calendar + content queue + one-click actions
